Cara edit website ini:

1. Buka file index.html dengan text editor (VS Code, Notepad++, Sublime, dll)
2. Edit teks langsung di dalam tag HTML
3. Ganti foto: taruh foto baru di folder "images/" dengan nama yang sama:
   - images/profile.jpg          (foto profil utama)
   - images/project-digital-twin.jpg, images/project-*.jpg (foto proyek)
4. Ganti CV: timpa file "Kirana-Pirandika-Ayuputri-CV.pdf" dengan PDF baru (nama harus sama)
5. Buka index.html di browser untuk melihat hasilnya (cukup double-click)

Tidak perlu install apapun - hanya butuh browser.
